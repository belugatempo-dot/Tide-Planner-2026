  运行命令

  npm install   # 安装依赖 (已完成)
  npm run dev   # 启动开发服务器 → http://localhost:5173
  npm run build # 生产构建 → dist/

  验证清单

  | 功能点         | 测试方法                                 |
  |----------------|------------------------------------------|
  | Wizard 流程    | 点击 Next 遍历 8 步，Back 返回上一步     |
  | 进度条         | 顶部显示 Step x / 8，进度百分比动态更新  |
  | 输入保存       | 填写内容后刷新页面，检查数据恢复         |
  | 步骤恢复       | 停在 Step 5 刷新，应恢复到 Step 5        |
  | 目标增删       | Step 6 添加/移除目标（最少1个，最多5个） |
  | 生成结果       | 完成 8 步后进入结果页，显示 Markdown     |
  | Copy Markdown  | 点击复制按钮，粘贴到任意文本编辑器验证   |
  | Download .md   | 点击下载，检查生成的 2026-annual-plan.md |
  | Print          | 点击打印，预览页面（隐藏按钮和进度条）   |
  | Edit Responses | 结果页点击 Edit，返回 Step 1             |
  | Start Over     | 点击 Start Over 并确认，清空所有数据     |
  | 移动端         | 缩小浏览器宽度，检查布局自适应           |

  功能亮点

  - 8 步 Wizard，覆盖 2025 回顾 → 2026 规划完整流程
  - 自动 debounce 保存到 localStorage（300ms 延迟）
  - 未填写字段在结果页显示 "(Not filled)"
  - 结果页支持三种导出：复制、下载、打印
  - 干净的 Tailwind UI，tide 蓝色主题w
