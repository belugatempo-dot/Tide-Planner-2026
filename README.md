# Tide Planner 2026

> Close 2025, set the tide for 2026.

A guided wizard to help you reflect on the past year and create a structured annual plan for 2026.

## Features

- **8-step wizard** covering review, patterns, direction, goals, and planning
- **Auto-save** to localStorage (never lose your progress)
- **Export options**: Copy Markdown, download .md file, or print
- **Responsive** design for desktop and mobile

## Wizard Steps

1. **2025 Review** - Significant events and achievements
2. **2025 Challenges** - Difficulties and lessons learned
3. **What Worked** - Effective habits and approaches
4. **Energy Patterns** - Sources and drains
5. **2026 Direction** - Key themes and "not doing" list
6. **2026 Goals** - 3-5 measurable objectives
7. **Quarterly Plan** - Q1-Q4 milestones and habits
8. **Risks & Support** - Obstacles and resources needed
